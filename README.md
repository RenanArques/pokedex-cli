# pokedex-cli
A simple CRUD Pokedex CLI developed in C

![Pokedex CLI](.github/pokedex-cli.png?raw=true "Pokedex CLI")

## Building

```bash
$ git clone https://github.com/rnqs/pokedex-cli.git

$ cd pokedex-cli && make

$ ./main
```
